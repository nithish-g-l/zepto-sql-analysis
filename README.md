# Zepto Inventory SQL Analysis
"LOVE BROTHER"
This project analyzes Zepto's inventory dataset using SQL to derive meaningful business insights. The analysis focuses on product pricing, discounts, category performance, inventory distribution, and other key metrics through structured SQL queries.

## Objectives
- Analyze product and category distribution
- Identify pricing and discount trends
- Evaluate inventory availability
- Generate business insights using SQL
- Practice advanced SQL concepts and query optimization

## Key Insights
- Category-wise product count and performance
- Top discounted products
- Pricing distribution across categories
- Inventory stock analysis
- Business recommendations based on query results

## Outcome
The project demonstrates how SQL can be used to transform raw inventory data into actionable business insights for retail and e-commerce decision-making.
